# rust-pixelator
A rust program that pixelates images. Currently only works with PNG images in testing.

## Build

Clone repository

`git clone https://github.com/MyYogurt/rust-pixelator.git`

`cd rust-pixelator`

`cargo build --release`

## Usage

`rust-pixelator <path to input image> <desired output path>`
